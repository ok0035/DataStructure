# DataStructure
자료구조 공부용 레포지토리

## Stack
Mutable List를 사용해서 간단하게 만든 Stack

```
    val stack = Stack<String>()
    stack.add("1")
    stack.add("2")
    stack.add("3")
    stack.add("4")
    stack.pop()
```

## Queue
Mutable List를 사용해서 간단하게 만든 Queue

```
    val queue = Queue<String>()
    queue.enqueue("1")
    queue.enqueue("2")
    queue.enqueue("3")
    queue.enqueue("4")
    queue.dequeue()
```
